A web application that helps users learn new languages through AI-powered lessons, interactive quizzes, and personalized learning paths. Built with Next.js, Tailwind CSS, and shadcn/ui.

## Features

- AI-powered adaptive lessons tailored to each learner's level
- Support for 30+ global languages
- Progress tracking with detailed analytics
- Google authentication for easy sign-in
- Responsive design for desktop and mobile

## Tech Stack

- **Framework:** Next.js 16 (App Router)
- **Styling:** Tailwind CSS 4 + shadcn/ui
- **Animations:** Framer Motion
- **Language:** TypeScript
- **Backend:** Firebase

## Getting Started

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
