# BLLP-AI — Bilingual Language Learning Platform Powered by AI

A modern web application that helps users learn new languages through structured, interactive courses. Built with Next.js, Tailwind CSS, and Firebase Authentication.

## Features

- 12 language courses with 15 lessons each (Spanish, French, German, Japanese, Arabic, Turkish, Italian, Portuguese, Serbian, Mandarin, Hindi, Russian)
- User authentication via email/password and Google sign-in
- Course enrollment and progress tracking per user
- Responsive dashboard with mobile-friendly sidebar
- Beautiful, accessible UI built with shadcn/ui components

## Tech Stack

- **Framework:** Next.js 16 (App Router)
- **Styling:** Tailwind CSS 4, shadcn/ui
- **Auth:** Firebase Authentication
- **Language:** TypeScript
- **Testing:** Vitest, React Testing Library
- **Animations:** Framer Motion

## Getting Started

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
